# 接亲小游戏 - 技术文档

## 项目概述
一款浪漫求婚风格的桌面接亲小游戏，玩家通过完成三个任务获取钥匙，最终打开宝箱。

## 技术栈
- **后端**：Python 3 + PyQt6（窗口控制、状态管理、TinyCC C++编译）
- **前端**：HTML/CSS/JavaScript（QWebEngineView 嵌入）
- **C++运行时**：TinyCC（打包进 exe，无编译器也能运行）
- **打包工具**：PyInstaller

## 目录结构
```
project/
├── py/

```

## 任务设计

### 任务一：打字游戏
- 显示一段浪漫文字
- 玩家在60秒内打完
- 要求速度 > 70字/分钟
- 完成后获得钥匙1，按 T 键跳过

### 任务二：C++ 算法题
- 显示题目：用 C++ 设计算法输出 2025.6.14
- 提供代码编辑器（预填充框架）
- 使用 TinyCC 编译运行
- 正确输出 2025.6.14 即通过
- 按 C 键跳过

### 任务三：看图答题
- 显示 imgs/任务三.png 图片
- 玩家输入答案
- 正确答案是 5201314
- 按 S 键跳过

## 快捷键设计
| 键 | 效果 |
|---|---|
| T | 任务一跳过，直接获得钥匙1 |
| C | 任务二跳过，直接获得钥匙2 |
| S | 任务三跳过，直接获得钥匙3 |

## 流程图
```
欢迎页 → 任务一 → 任务二 → 任务三 → 宝箱页 → 爱心展示 → 最终问题
```

## 打包命令
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --add-data "assets;assets" --add-data "imgs;imgs" py/main.py
```
